操作指南

◆开始游戏首次进入需要填写 DeepSeek API Key。保存后系统会自动继续接入直播。
◆行动输入你可以点击选项，也可以在底部输入框自由描述行动、对话、交易、逃跑、结盟或使用能力。
◆身份 ROLL如果随机身份不满意，可以选择“重新 ROLL”。重新 ROLL 不会推进剧情，也不会惩罚玩家。
◆S7 存活如果玩家在 S7 中存活，可以输入“继续到下一场”或选择对应选项，让主持人开启下一场秀场。
◆状态面板菜单中的状态面板显示腕带数据。手机端会以顶部浮窗打开，右上角 × 可关闭。
◆主题与回顾主题可切换全局颜色；回顾可查看以往剧情与选择；更新日志记录版本变化。
